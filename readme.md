# In this project, we need WSL or Linux 

# Update system and install Python
```bash
sudo apt update && sudo apt upgrade -y
sudo apt install python3 python3-pip -y
```
# RUN
```bash
python3 app.py
```
# Access to the WEBSITE
Access at: http://localhost:5000
